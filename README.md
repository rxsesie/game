# Cozy Forest Explorer

A visually rich Kaboom.js prototype inspired by cozy forest dioramas and soft storybook worlds.

## Run locally

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

## Controls

- Move: `WASD` / Arrow keys
- `E`: Rest at campfire (outfit shuffle)
- `H`: Pet companion
- `T`: Water/trim bonsai
- `F`: Fish and release into pond
- `R`: Breathing corner ritual
- `J`: Open journal overlay

## Added polish pass

- Live style customizer in HUD (skin, outfit, pet)
- Discovery progress chip and richer status metrics
- Layered terrain and decorative path painting
- Fireflies + weather overlays for ambience

Progress persists via `localStorage`.
