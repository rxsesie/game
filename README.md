# Cozy Forest Explorer

A visually rich Kaboom.js prototype inspired by cozy forest dioramas and soft storybook worlds.

## Run locally

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

## Controls

- Move: `WASD` / Arrow keys
- `E`: Rest at campfire (style refresh)
- `H`: Pet companion
- `T`: Water/trim bonsai
- `F`: Fish and release into pond
- `R`: Breathing corner ritual
- `J`: Open journal overlay

## Current visual direction

- Layered terrain palette with meadow / forest / cliff / water biomes
- Rich decorative prop pass (trees, rocks, flowers, lantern glow)
- Dynamic day-cycle tint and weather (mist + drizzle)
- Ambient glows, shadows, and cozy campsite set pieces

Progress persists via `localStorage`.
